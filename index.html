MQT_GlobexHighLowShading

Visualize Globex Session Ranges — Instantly

Automatically shade the full Globex session's high-to-low range across all Globex bars. Configurable time window (default 5:00 PM → 8:30 AM), adjustable opacity, and custom color. Built for futures traders on NinjaTrader 8 who need clear visual separation between Globex and RTH.

Key Features

Full Session Range Shading

Shades from the session's highest price to lowest price across all bars within the Globex window.

Configurable Time Window

Set Globex Start and Globex End times. Handles overnight sessions that cross midnight.

Adjustable Opacity

Control transparency from 1–100% to keep your chart readable while highlighting the Globex range.

Custom Shade Color

Choose any color via the Shade Brush property. Default is gray for subtle contrast.

Direct2D Rendering

Uses SharpDX for smooth, hardware-accelerated fills — no performance impact on large datasets.

Real-Time Updates

Calculates on each tick (OnEachTick) and retro-fills finalized sessions for historical accuracy.

How It Works

1

Detect Globex Bars

Checks each bar's timestamp against your configured time window.

2

Track Session Extremes

Maintains running high/low for the current Globex session.

3

Finalize on Exit

When the session ends, retro-fills all bars with the final high/low.

4

Render Shaded Bands

Uses Direct2D to fill rectangles from high to low for each visible bar.

FAQ

Does this work for overnight sessions?

Yes. The indicator handles time windows that cross midnight (e.g., 17:00 → 08:30).

Can I change the shaded color?

Yes. Use the Shade Brush property to select any color.

Will it slow down my chart?

No. Direct2D rendering is hardware-accelerated and highly efficient.

Does it repaint?

No. Once a session ends, the high/low is finalized and retro-filled.

Can I use this on any timeframe?

Yes — minute, tick, second, or daily charts all work.
